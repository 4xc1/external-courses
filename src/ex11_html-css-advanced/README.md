## Задание 1

Создайте три анимации:

-   лоадер для страницы

-   шар, прыгающий между двумя блоками

-   бесконечная анимация со следующими шагами:

    -   круг с градиентом

    -   квадрат

    -   смена градиента

    -   изменение размера и искажение

    -   вращение

    -   изменение градиента

    -   круг
